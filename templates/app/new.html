Adding New Applications

Applications are software repositories hosted on Github which are used by the system to create new Wisps for users. Wisps are used by Appliances to launch Instances with the deployed application after noting a payment to a given Bitcoin Address.

To create a new application, past the URL of a Public Repository below and then click Add Application.

Note: You may want to start by forking a sample repository or adding the launcher badge to an existing application's repository.
