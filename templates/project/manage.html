Modify the Project's Repository

The repository this project points to needs to be configured to allow {{ app_name }} to start instances for users. Use the following commands to add the utterio directory structure to this repository:

git clone {{ project.url }}
cd {{ project.repo_name }}
curl -L http://goo.gl/avqPgq > utterio.tar.gz
tar xvfz utterio.tar.gz
mv gist* utterio
rm utterio.tar.gz
git add .
git commit -m 'adding utter.io cloud capabilities'
git push

Once you are done adding the directory to your repository, click the Sync Project w/Github button below.
